# IT 팀 빌딩 서비스 FE(서비스 이름 미정)

## 실행 방법

node 16

```bash
npm install
-- husky 설치
npm run prepare
-- 실행 (http://localhost:5173)
npm run dev
-- story book 실행 (http://localhost:6006)
npm run storybook
```
